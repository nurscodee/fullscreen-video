# Fullscreen Video Background Project

A clean, minimalist web page featuring a responsive fullscreen video background with interactive controls. This project was built to practice foundational front-end development skills, focusing on semantic HTML, modern CSS layouts, and vanilla JavaScript DOM manipulation.

## 🧠 What I Learned (Learning Outcomes)

This project was a fantastic learning experience where I successfully implemented and mastered several core web development concepts:

- **CSS Transitions:** Learned how to create smooth user experiences by applying `transition: background-color 1.2s;` to the control button, making the hover state feel modern and fluid.
- **JavaScript Video Controls:** Understood how to interact with HTML5 video elements using vanilla JS, specifically utilizing the `.play()` and `.pause()` methods to control media playback dynamically.
- **Conditional Logic (if/else statements):** Practiced implementing conditional structures in JavaScript to check the current state of the video (`video.paused`) and toggle the button text and video playback accordingly.
- **UI/UX Best Practices:** Enhanced accessibility and user feedback by adding the `cursor: pointer;` property to interactive elements.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure (`<main>`, `<section>`, `<video>`).
- **CSS3:** Fixed positioning, custom buttons, transitions, and hover effects.
- **JavaScript (ES6):** Event listeners, conditional logic, and DOM manipulation.

## 📂 Project Structure

```text
├── index.html          # HTML structure and video element
├── style.css           # Styling, transitions, and layout
├── script.js          # Video playback and button toggle logic
└── videos/
    └── background-video.mp4  # Asset used for the background
```

## ⚙️ How to Run the Project

1. Clone or download this repository to your local machine.

2. Ensure you have the video file placed in the correct directory (/videos/background-video.mp4).

3. Open the index.html file in any modern web browser.

4. Click the Pause/Play button to control the background video.
